Daily COVID-19 Updates (Global)

This dataset is the aggregate of all the CSV files provided by COVID-19 Data Repository by the Center for Systems Science and Engineering (CSSE) at Johns Hopkins University. The folder on its own is hundreds of CSV files, but this dataset cuts down processing time for exploring the data as a whole.

Twitter Trolls & Bots

An incomplete list of Twitter accounts analyzed by Bot Sentinel. At this time, Bot Sentinel does not have a public API to retrieve their data, so I made a web scraping script using Selenial to collect 2,000 accounts once a week. In addition to data from Bot Sentinel, I added additional features using Twitter's API.

COVID-19 in Prisons/Jails

This dataset is the aggregate of updates provided by UCLA Law. In its early stages, the updates were only available from a Google Sheet, so I worked with some of the projects members to create an aggregated version of their data. This data was used in my analysis of policing during the early stages of the pandemic.

Venue Details From Resident Avisor (Global)

Here, I used web scraping to get venue information from all of the clubs that have pages on Resident Advisor. The pages do not explicitly give all the details provided, and application of Natural Language Processing was used to find out if a venue hosts All-Ages events, along with the number of stages a venue might have. Future enhancements of this dataset will include geographic location.

Event Details From Resident Avisor (Global)

A useful dataset that compiments the Venue Details from Resident Advisor. Here you can find event informoation, between 2010 and 2020 from most of the venues that have Resident Advisor pages. This is for an ongoing project that will be used to analyze distribution of gigs between independent, DIY, and major signed artists.
